Project Description

In this project, we implement a hierarchical text classification model. The data we use is 20news dataset. There are 7 general labels and 15 sub-labels. The model we use is CNN. After training process, we can give each text one general label and one sub-label.
We have 10000 training data and 7000 test data. The label distribution is showed below.

Model Review

The model consists of 2 layer of CNNs. After the first layer of CNN consisted of three different kernel size channels, the model outputs the first layer label. Then the features go through the second similar CNN layer, and outputs the second layer label.
The embedding is also a fully connected layer, which gives each text an embedding vector of 128 dimensions. The second fully connected layer is 128*128, which gives the output of the first general label. The second fully connected layer is 128*50, which gives the output of the sub-label.
